

*Portfolio Website



This is a simple portfolio website created by Hrushikesh Punem to showcase personal information, projects, and contact details. The website is built using HTML,  CSS,  and JavaScript.

*Description


This portfolio website was developed as a project assignment for the Computing Lab subject at NITK, as part of the CSIS department's curriculum. It serves as a platform to introduce Hrushikesh Punem and display essential information such as a brief bio, educational background, skills, projects, and contact details. The website also includes links to external profiles like LinkedIn, GitHub, and Instagram.


*Features


Home section with tabs for education, project, certification, coursework, hobbies, info.
Education section with all education details.
Projects section with project details.
Certification section containing certifications.
Coursework section with all the course and learning details.
Hobbies section containing all the hobbies.
Contact Me section with contact information and links to social profiles
